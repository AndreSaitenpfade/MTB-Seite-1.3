# Core // Components - Bike Projekt

Willkommen beim offiziellen Repository für **Core // Components**. Diese Webseite präsentiert hochwertige Mountainbike-Komponenten mit Fokus auf Trail-Performance.

## 🚀 Features
- Responsives Design (optimiert für Desktop und Mobile).
- Interaktive Athleten-Sektion.
- Support-Modal zur schnellen Kontaktaufnahme.
- Social-Media-Integration.

## 🛠 Tech-Stack
- **HTML5**
- **CSS3** (mit Flexbox & Media Queries für responsives Layout)
- **JavaScript** (für die interaktiven Elemente)

Du kannst die Webseite hier live sehen:

## 📂 Struktur des Repositories
- `/images`: Alle Bilddateien (Icons, Logo, Produktbilder).
- `index.html`: Hauptstruktur der Webseite.
- `style.css`: Alle Design-Anpassungen.
- `script.js`: Logik für Buttons und Modals.
